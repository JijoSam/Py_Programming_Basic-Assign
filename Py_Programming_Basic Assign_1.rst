4. Write a Python program to swap two variables?
================================================

.. code:: ipython3

    x = 5
    y = 4
    
    x , y = y , x
    
    print("x swaped to",x)
    print("y swaped to", y)


.. parsed-literal::

    x swaped to 4
    y swaped to 5
